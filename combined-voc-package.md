# 상품팀 VOC 개선 패키지

## 1. 상품팀 VOC 응대 지원 AI 에이전트 기획안

### 목적
반복되는 상품 VOC에 대해 기존 답변을 더 빠르고, 더 일관되게 재사용할 수 있도록 지원한다.

### 1. 문제정의
- 반복 VOC가 지속 유입되지만 답변 이력이 분산됨
- 시스템, FAQ, 개인문서에 흩어져 재사용이 어려움
- 자산화되지 않아 유사 답변을 반복 작성함
- 담당자별 작성 시간과 답변 일관성 편차가 발생함

### 2. 대상 사용자
- 상품팀 VOC 답변 담당자
- 현장 담당자

### 3. 핵심 기능
- 문의 유형 자동 분류
- 유사 VOC / 과거 답변 검색
- 매뉴얼 · 약관 · 상품문서 추천
- 시스템 조회 체크리스트 제공
- 근거 기반 답변 초안 생성

### 4. 안 만들 것
- 계산 근거 없는 추론 답변
- 승인 없는 자동 발송
- 전 상품 · 전 업무 범용 자동화
- 계산 시스템 대체

### 5. 완료 정의
- 한 흐름에서 분류 · 검색 · 근거 · 초안 제공
- 우선 대상 VOC 처리 시간 단축
- 담당자 간 품질 편차 감소
- 최종 답변은 사람 검토 후 사용

### 입력/출력 흐름
1. 입력: VOC 문의 내용 접수
2. 검색·분류: 유사 VOC / 과거 답변 탐색
3. 근거 확인: 문서 · 표준답변 · 체크리스트 확인
4. 출력: 답변 초안 생성
5. 검토·승인: 담당자 최종 확인

### 운영 원칙
모든 답변은 사람 검토 후 활용하며, AI는 반복 답변 재사용과 초안 작성을 지원한다.

## [별첨] 현상 문제점 및 개선방안 요약

### 현상 문제점

#### 메인프로젝트
- 반복 VOC 답변 재사용 안 됨
- 답변 이력 분산
- 유사 답변 반복 작성

#### 서브프로젝트
- 현장의 담당 부서 판단 어려움
- 부서간 R&R 불명확
- 담당자 경험 의존

### 개선방안 요약

#### 메인프로젝트
- 반복 상위 유형 표준 답변 자산화
- 현장 사전조회 후 협조요청
- 상품팀 기존 답변 참고·보정

#### 서브프로젝트
- 부서별 R&R 명확화
- 현장 분류 기준 정비
- 재배정·재확인 감소

### 기대효과
- 월 평균 200건 중 반복 유형 약 30%를 1차 개선 대상으로 설정
- 현장 자체 해결 가능 문의 확대
- 상품팀 건당 답변시간 단축 및 답변 일관성 향상
- 장기적으로 부서 분류 정확도 개선 기반 확보

### 측정 방법
- 월 VOC 협조요청 건수 추이
- 반복 유형 협조요청 비중 변화
- 표준 답변 사용률 및 상품팀 건당 답변시간
- 재배정 건수 / 재확인 요청 건수

### 핵심 방향
메인프로젝트로 단기 효과를 먼저 확보하고, 서브프로젝트로 분류 기준 정비를 연계 추진한다.

## [별첨] AS-IS 문제 구조

### 현재 프로세스
1. 고객 문의
2. 현장 접수
3. 자체 해결 가능 여부 판단
4. 자체 해결 불가 시 VOC 협조요청 입력
5. 담당 부서 자동 배정
6. 담당 부서 답변 작성
7. VOC 시스템 입력 완료

### 메인프로젝트: 반복 답변 비재사용
- 유사 문의 반복 발생
- 답변 이력 분산
  - 시스템 / FAQ / 개인문서
- 유사 답변 반복 작성

### 서브프로젝트: 현장 분류 실패
- 답변 부서 찾기 어려움
- 부서간 R&R 불명확
- 담당자 경험 의존

### 핵심 메시지
현장은 부서 판단이 어렵고, 상품팀은 같은 유형의 답변을 반복 작성하는 구조다.
