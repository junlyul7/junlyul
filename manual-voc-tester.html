REVIEW POINTS

상담사는 답변 자체보다 근거와 위험 신호를 먼저 확인합니다.

1. 라우터 판단 입력 문장을 잡담, 보강 필요, 고객문의, 담당자 문의로 먼저 나눕니다.
2. 유사문 검색 `manual_docs.csv`의 embedding을 활용해 가까운 사례를 찾습니다.
3. 초안 제시 `standard_answer`를 기반으로 검수 가능한 초안을 보여줍니다.

ASK THE BOT

현장 테스트용으로 질문을 넣고, 라우터 분기 / 검색 / 초안을 바로 확인하세요. 배포 전 구성을 검토하는 목적이라 로컬 페이지에서 바로 실행됩니다.

브라우저에서 바로 호출하는 테스트 화면입니다. CORS는 `manual-voc-flow` 함수에 이미 설정되어 있습니다. Endpoint와 Key는 자동 저장됩니다.
